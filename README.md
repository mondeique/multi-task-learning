# multi-task-learning
MONDEIQUE multi task learning (MTL) category classification using Tensorflow

## How to install 
```
pip install -r requirements.txt
```
## How to run 
```
run cell tf_MTL.ipynb
```
## Data
1. Images
2. CSV
## Data Structure
```
|
|-- data
      |-- images
            |-- training (training/evaluation)
            |-- test (test images)
      |-- csv
            |-- training
            |-- test
|-- split_csv.ipynb
|-- tf_MTL.ipynb (main training file)
```
